# 虚拟机申请列表添加数据盘展示列

## 背景

左侧菜单“虚拟机申请”对应的页面为 `CreateRequest.vue`。该页面中的“我的虚拟机申请列表”表格已展示系统盘、总数据盘、总磁盘等信息，但未直接展示每节点的数据盘大小，用户在列表中无法直观查看单台虚拟机的数据盘配置。

## 目标

在“我的虚拟机申请列表”表格中增加“数据盘(GB)”列，展示每条申请记录中每节点的数据盘大小。

## 方案

在现有表格的“系统盘(GB)”列之后，新增“数据盘(GB)”列：

- 字段来源：`row.data_disk`
- 空值处理：当 `data_disk` 为空或未定义时，显示为 `-`
- 列宽：100px，居中对齐
- 保留现有“总数据盘(GB)”和“总磁盘(GB)”列不变

## 影响范围

- 前端文件：`frontend/src/views/CreateRequest.vue`
- 仅涉及表格列定义，不改变数据模型、API 或业务逻辑

## 验收标准

- [ ] 列表中新增“数据盘(GB)”列，正确显示每节点数据盘大小
- [ ] 数据为空时显示 `-`
- [ ] 现有“总数据盘(GB)”和“总磁盘(GB)”列仍然正常展示
- [ ] 页面布局无异常，表格横向滚动正常
